Este repositorio contiene una colección de proyectos personales organizados por categorías:

### 🏗️ Proyectos Docker
- **`dockerfile-projects/`**: Colección de proyectos containerizados
  - **`CRM Project/`**: Sistema CRM completo con SuiteCRM, MySQL y Apache
  - **`ai/`**: Proyectos de inteligencia artificial
  - **`back-end/`**: APIs y servicios backend
  - **`front-end/`**: Aplicaciones web modernas

### 📚 Notas y Documentación
- **`notes/`**: Notas personales y documentación técnica

- **`windows/`**: Herramientas y scripts para entornos Windows
